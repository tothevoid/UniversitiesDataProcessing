Методы для обработки данных студентов

Модули:
    age_determiner.py:
        Определение возраста человека посредством его друзей. Используется в качестве dataset использутеся csv с распределением количества друзей по промежуткам.
    filter.py:
        Функции:
        union
            Подготавливает выборку для обчуения нейросети. К распределению возрастов добавляется возраст, указанный пользователем (если он есть) из изначальной выборки.
        split_files
            Разделяет файл на обучающую выборку (с известным возрастом) и валидационную выборку (с неизвестым возрастом)
    helpers.py:
        Набор часто используемых функций 
        Функции:
        save_file:
            Сохраняет список словарей в файл
            Аргументы:
            output_name - название сохраняемоего файла
            output_items - список словарей для сохранения в csv
        read_csv_as_dict
            Загружает csv-файл в виде списка словарей
            Аргументы:
            relative_path - относительный путь к файлу 
            delimeter - разделитель в csv
            Возвращает: список словарей полученный из csv
    vk_data_to_csv.py:
        Парсер данных, полученных из vk посредством метода execute. 
        file_parse_bdays:
            Аргументы:
            json_string - json с информацией о друзьях пользователей, полученный функцией get_freinds в модуле vk_parser 
        file_parse_cities:
            Аргументы:
            json_string - json с информацией о друзьях пользователей, полученный функцией get_freinds в модуле vk_parser 
    vk_data_to_csv_legacy.py:
        Функции для обработки данных в устаревшем виде
    vk_parser.py:
        Модуль для обогащения данных, посредством выгрузки списка друзей (с дополнительной информацией) из vk
        get_freinds:
            Аргумент: вектор id пользователей, друзей которых необходимо получить
        save_results:
            Сохраняет промежуточные результаты в файл с номером текущей итерации
            Аргументы:
            name - имя сохраняемого файла
            results - список выгруженных и преобразованных данных вк
        vk_execute:
            Генерация и выполнение кода в методе execute для vk_api
            Аргументы:
            script - скрипт из script.js
            ids - ветор id пользователей, друзей которых, нужно получить
            Возвращает:
            Результат скрипта в виде json
Дополнительные файлы:
    script.js - код для метода execute в vk_api. На месте {{id}} поставляются id пользователей, друзей которых нужно получить

Схема работы:
    1. Есть изначальная выборка со студенами вуза в формате csv
    2. Функция get_freinds (модуль vk_parser) для получения json-файла со списками друзей(с информацией о них)
    3. Функции file_parse_bdays/file_parse_cities (модуль vk_data_to_csv) для трансформации данных и перевода в csv
    4. Функции union и split_files для преведения к файлам для обучения модели
    5. Полученные данные используются в модуле age_determiner, где нужно определить возраст и местоположение
