Методы для обработки данных студентов

# Модули:
## `age_determiner.py`:
    Определение возраста человека посредством его друзей. В качестве dataset использутеся csv с распределением количества друзей по промежуткам.
## `age_model.py`:
    Скрипт c использованием готовой модели для определения возраста.
## `ages_frequency.py`:
    Скрипт для определения распределения частот возрастов в выборке.
## `coords.py`:
    Скрипт для определения распределения частот возрастов в выборке.
#### `get_coords`: Получает и сохраняет координаты csv
##### Аргументы:
* university - название вуза в папке datasets
#### `load_coords`: Получает координаты центра населенного пункта через OpenStreetMap.
##### Аргументы:
* places - Список названий мест, координаты которых, нужно получить
#### `get_regions`: Получает координаты контура регионов и вовращает их в csv.
##### Аргументы:
* name - имя файла c названиями регионов
##### Возвращает:
* Сsv-файл вида ('Название', 'Определённое название', 'Набор координат контура')
## `helpers.py`:
    Набор часто используемых функций 
#### `save_file`:
Сохраняет список словарей в файл
##### Аргументы:
* output_name - название сохраняемоего файла
* output_items - список словарей для сохранения в csv
#### `datetime_parse`:
Переведит дату рождения в количество лет (если это возможно)
##### Аргументы:
* dt - дата 
##### Возвращает: список словарей полученный из csv
## `migrations.py`:
    Скрипт для сбора и сохрания в csv, информации о миграции студентов
#### `format_json`:
Конвертирует json из полей `universities` и `schools` в название города (при возомжности, в хронологическом порядке)
##### Аргументы:
* string - json необходимый конвертировать
#### `get_migrations`:
Переведит дату рождения в количество лет (если это возможно)
##### Аргументы:
* university - название университа в папке `datasets`
* univercity_city - город, в котором находится университет (например 'Москва')
* is_schools - булевое значение  
    Варианты:
    * True - выгружает миграции из школы в университет
    * False - выгружает миграции после окончания университета
##### Возвращает: список словарей полученный из csv
## `run_methods.py`:
    Скрипт для последовательного выполнения функций по всем университетам
## `unique_cities.py`:
    Скрипт для последовательного выполнения функций по всем университетам
#### `find_cities`:
Получает все уникальные названия городов из полей `universities` и `schools`
##### Аргументы:
* uni - название университета в папке `datasets`
##### Возвращает:  
* Список названий уникальных городов
#### `get_cities`:
Получается все уникальные названия городов по всем вузам
##### Аргументы:
* universities - список названий университетов из папки `datasets` 
## `vk_freinds_json_parser`:
    Модуль для обработки друзей выпускников, загруженных модулем  `vk_friends_loader.py`
#### `file_parse_bdays`:
Получает распределение возрастов друзей выпускников
##### Аргументы:
* json_string - json-строка, полученная с помощью `vk_friends_loader.py` 
##### Возвращает:  
* result_male - распределение возрастов по мужскому полу
* result_female - распределение возрастов по женскому полу
* result_all - распределение возрастов
#### `create_ranges_dict`:
Генерирует словарь с различными интервалами возрастов, по которым будут строится распределения в функции `file_parse_bdays`
##### Возвращает:
* Словарь интервалов возрастов
#### `quantify`:
Распределяет друзей выпускника по словарю, полученному с помощью функции `create_ranges_dict`:
##### Аргументы:
* friends_list - список словарей друзей выпусукника
* root_id - id выпускника
* root_sex - пол выпускника
* root_bdate - возраст выпускника
##### Возвращает:  
* Словарь (распределением друзей выпускника по интервалам возрастов и id, пол, возраст выпускника)
#### `file_parse_cities`:
Собирает топ-5 городов, в которых проживают друзья выпускника.
##### Аргументы:
* json_string - json-строка, полученная с помощью `vk_friends_loader.py` 
##### Возвращает:  
* Распределение городов в которых проживают друзья выпускника.
#### `parse_json_files`:
Функция для парсинга всех файлов, полученных с помощью модуля `vk_friends_loader.py`. Создает 7 csv файлов с различными распределениями.
##### Аргументы:
* university - название университета в папке `datasets`
#### `quantify_ages`:
    Переводит реальный возраст пользователя в шкалу начинающуюся с 0
##### Аргументы:
* dataset - выборка выпускников
##### Возвращает:  
* Выборку выпускников с изменным возрастом в определенном диапазоне
## `vk_friends_loader.py`:
    Модуль для обогащения данных, посредством выгрузки списка друзей (с дополнительной информацией) из vk
#### `load_data`:
Выгружает и сохраняет всех друзей пользователей в json вместе с информацией о самом пользователе
##### Аргументы:
* data - dataframe с колонками ('id','bdate','sex'), пользователей, друзей которых, нужно выгрузить
#### `save_results`:
Сохраняет промежуточные результаты
##### Аргументы:
* name - название файла
* results - список словарей выгруженных пользователей
#### `vk_execute`:
Генерация и выполнение кода в методе execute для vk_api
##### Аргументы:
* script - скрипт на vkscript в виде строки
* ids - строка из id пользователей, который перечислены через запятую (Например: `'1213552,213242,521254,2542505'`)
##### Возвращает:
* json-ответ от api vk
#### `get_freinds`:
Собирает всех выпускников вуза и выгружает их друзей 
##### Аргументы:
* university - название университета из папки `datasets`
# Дополнительные файлы:
* script.js - код для метода execute в vk_api. На месте {{id}} поставляются id пользователей, друзей которых нужно получить
* cities.json - города с id по базе vk 
* countries.json - страны с id по базе vk 
# Основная схема работы:
1. Есть изначальная выборка со студенами вуза в формате csv
2. Выгружается информация о друзьях выпускников с помощью api vk
3. Данные парсятся и собирается выборка для обучения
4. Полученные данные используются в модуле age_determiner, где нужно определить возраст и местоположение
